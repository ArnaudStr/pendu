Petit jeu du pendu codé en C.
Rien de bien compliqué, si vous avez GCC d'installé vous pouvez compiler avec la commande : gcc -o pendu pendu.c dico.c
et executer avec la commande ./pendu puis suivez les instructions sur le terminal :).
Le mode 1 joueur permet de jouer seul avec un mot choisi aléatoirement dans dictionnaire.txt et le mode 2 joueurs permet à 1er joueur d'entrer un mot et à un 2eme joueur de trouver de quel mot il s'agit.
